# Candidate Management App

A simple Candidate Management application built using Next.js (App Router).  
The app allows users to add, view, and delete candidates with basic validation and clean UI.

---

## Features
- Add candidates (name, email, role)
- View candidate list
- Delete candidates
- Data persists using localStorage

---

## Tech Stack
- Next.js (App Router)
- TypeScript
- React
- Tailwind CSS

---

## Setup

```bash
npm install
npm run dev
